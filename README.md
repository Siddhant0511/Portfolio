# Portfolio Project — Siddhant Morye

## Project Structure

* **[index.html](file:///e:/ALL%20CLAUDE/PORTFOLIO/index.html):** The core single-page portfolio containing all HTML structure, inline responsive Vanilla CSS styles, and client-side JavaScript for the interactive FLIP modal and page animations.
* **[DESIGN.md](file:///e:/ALL%20CLAUDE/PORTFOLIO/DESIGN.md):** Active design tokens, typography scales, layout rules, and motion details.
* **[_CONTEXT.md](file:///e:/ALL%20CLAUDE/PORTFOLIO/_CONTEXT.md):** Ongoing handoff context and implemented milestones.
* **[img/](file:///e:/ALL%20CLAUDE/PORTFOLIO/img/):** Avatar images and original assets.
* **[Siddhant_Morye_Resume.pdf](file:///e:/ALL%20CLAUDE/PORTFOLIO/Siddhant_Morye_Resume.pdf):** Standalone PDF copy of the resume.
